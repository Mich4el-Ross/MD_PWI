JQuerry


O JQuerry é uma é uma biblioteca de funções JavaScript que interage com o HTML, que foi desenvolvida para simplificar os scripts interpretados no navegador do cliente. Ela é uma biblioteca de código aberto na qual possui uma sintaxe simples a fim de facilitar a navegação do documento HTML. Basicamente ela compacta várias linhas de código em uma única função.
